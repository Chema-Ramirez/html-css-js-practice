PC

Definitions

  • Personal Computer: An electronic machine capable of storing information and automatically processing it using mathematical and logical operations controlled by computer programs.

  • Compatible Computer: A computer capable of running the same programs as other computers of the same type.

  • Desktop Computer: A desktop computer that includes a central unit, keyboard, display, and one or more disk drives and can operate without being connected to a computer network.

  • Notebook Computer: A small, lightweight personal computer that can be easily carried from one place to another, like a briefcase. A built-in battery allows for a certain amount of operating time.
